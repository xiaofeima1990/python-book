## 初始化：
git init

## 克隆远程文件项目到本地
git clone https:..../git

## 修完完添加
git add .

## commit 
git commit -m"note"

## 推送改动
git push origin master 

## 分支
创建一个叫做“feature_x”的分支，并切换过去：
git checkout -b feature_x

切回主分支
git checkout master

把新建分支删掉
git branch -d feature_x

推送到远程
git push origin <branch>
